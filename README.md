# Pre-Requirements

  - java 8
  - maven
  - postgresql
  - Docker (optional)

# Install and run API

 1. Clone repository
 2. CD to repository cloned
 3. Set up all enviroments variables
 4. Execute command "mvn install"
 5. Execute command "mvn spring-boot:run" for start API
